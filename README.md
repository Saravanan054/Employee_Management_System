# Employee Management System (Spring Boot Backend)

## Overview
Employee Management System (EMS) is a backend application built using Spring Boot.  
It provides RESTful APIs to manage employees, departments, projects, and other organizational data with full CRUD operations.

---

## Features
- Employee Management  
- Department & Designation Management  
- Project Management  
- Employee-Project Mapping  
- Attendance Tracking  
- Payroll Management  
- Leave Request Handling  
- User Management  

---

## Tech Stack
- Java  
- Spring Boot  
- Spring Data JPA (Hibernate)  
- MySQL  
- Maven
